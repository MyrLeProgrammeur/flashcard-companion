# flashcard-companion

Android companion app that replaces AnkiDroid as the review client for the decks
generated by [`flashcard-pipeline`](../flashcard-pipeline), with an AI explanation
button (Infercom, DeepSeek-V3.1) grounded in the card's source PDF.

- `backend/` — FastAPI server, runs **on the phone via Termux**, reads the `.apkg`
  files synced by Syncthing (read-only), holds its own SRS state (SM-2), serves the web UI.
- `android/` — minimal Kotlin WebView shell, points to `http://127.0.0.1:8420`.

See `docs/plans/` for the design history and ongoing decisions.

## Quick start (PC dev)

```bash
cd backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # set INFERCOM_API_KEY
uvicorn main:app --reload --port 8420
```

Open `http://localhost:8420/` in a browser to iterate on the UI without the phone.

## Deployment on the phone (Termux)

```bash
pkg install python
pip install -r backend/requirements.txt
bash backend/termux/start.sh
```

See `backend/termux/boot/` for auto-start on reboot (Termux:Boot, optional).
